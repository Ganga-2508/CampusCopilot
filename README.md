# CampusCopilot

AI-powered multi-agent career assistant for students and freshers.

## Problem Statement

Many students struggle with resume building, ATS optimization, career planning, skill gap identification, internship applications, and interview preparation.

## Solution

CampusCopilot is an AI-powered multi-agent platform that helps students improve their resumes, identify missing skills, generate personalized career roadmaps, and prepare for interviews.

## Core Agents

### Resume Analyzer Agent

* Analyze resumes
* Calculate ATS score
* Suggest improvements

### Skill Gap Agent

* Compare skills with target role
* Identify missing skills

### Career Roadmap Agent

* Generate 30-day and 60-day learning plans
* Recommend courses and projects

### Interview Coach Agent

* Generate interview questions
* Provide preparation guidance

## Target Users

* College Students
* Freshers
* Internship Seekers
* Engineering Students
